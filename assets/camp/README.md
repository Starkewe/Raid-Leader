# Camp V1 visual assets

`camp_facilities_atlas.png` and `camp_terrain_atlas.png` are original 4×3 modular atlases generated specifically for Raid Leader from the user-approved Camp mood reference. They intentionally do not contain third-party game-pack assets.

- Native atlas dimensions: 1448×1086 pixels each.
- Cell dimensions: 362×362 pixels.
- Runtime filtering: nearest-neighbor through project defaults and each Camp sprite.
- Usage: individual facility regions and repeated terrain regions only; never a flat painted Camp background.
- Projection goal: overhead gameplay footprints and navigation remain authoritative. The art can be replaced without changing facilities, saves, or member AI.

The source mood image is a design reference and is not distributed as an in-game background.
